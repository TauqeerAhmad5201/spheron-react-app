# My Spheron App - Documentation

## Quick Start
```bash
git clone [repository-url]
cd my-spheron-app
npm install
npm run build
npm start
```

```
# Start dev server
npm start

# Build for production
npm run build

# Docker commands
docker build -t my-spheron-app .
docker run -p 3000:3000 my-spheron-app
```
